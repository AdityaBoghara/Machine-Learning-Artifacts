<h1 id="linear-regression-interview-ready-summary">Linear Regression (Interview-Ready Summary)</h1>
<h2 id="overview">Overview</h2>
<p>Linear regression is a <strong>supervised learning</strong> algorithm used to model the relationship between a <strong>continuous dependent variable</strong> and one or more <strong>independent variables</strong> by fitting a linear function to observed data. The goal is to learn parameters that produce predictions with minimal error.</p>

<hr>
<h2 id="when-convergence-is-achieved">When Convergence Is Achieved</h2>
<p>Convergence occurs when:</p>
<ul>
<li>The change in the cost function becomes negligible, or</li>
<li>Parameter updates approach zero</li>
</ul>
<p>At this point, the model has reached the <strong>global minimum</strong> of the cost function.</p>
<hr>
<h2 id="intuition">Intuition</h2>
<ul>
<li>Gradient descent moves parameters toward the <strong>steepest decrease</strong> in error</li>
<li>The learning rate controls <strong>speed vs. stability</strong>
<ul>
<li>Too large → overshooting or divergence</li>
<li>Too small → slow convergence</li>
</ul>
</li>
</ul>
<hr>
<h2 id="final-outcome">Final Outcome</h2>
<p>At convergence:</p>
<ul>
<li>The cost function is minimized</li>
<li>Parameters are stable</li>
<li>The best-fit line (or hyperplane for multiple features) is obtained</li>
</ul>
<p>This approach scales efficiently to large datasets and extends naturally to <strong>multiple linear regression</strong> using vectorized operations.</p>
